# تطبيق سواعد عرب HR - Android WebView

تطبيق أندرويد بسيط (WebView) يعرض موقع `https://hr.sawaedarab.com/` كأنه تطبيق أصلي.

## المميزات المضمّنة
- عرض الموقع داخل WebView مع شريط تقدّم أعلى الشاشة.
- **رفع الملفات**: يدعم رفع المستندات والصور (مطلوب لصفحات مثل رفع إثبات التحويل، المستندات، وتسجيل الوجه بالكاميرا).
- **الكاميرا**: صلاحية الكاميرا مع FileProvider لدعم صفحة تسجيل الوجه (face-api.js) والتقاط صور/فيديو إثبات العمل الإضافي.
- **الموقع الجغرافي (GPS)**: صلاحية الموقع لدعم صفحات اعتماد العمل الإضافي التي تعتمد على GPS reverse geocoding.
- **تنزيل الملفات**: أي رابط تنزيل (تصدير PDF/Excel من صفحات الرواتب، الحضور، إلخ) يُنزَّل عبر مدير التنزيلات في أندرويد ويظهر إشعار عند الانتهاء.
- **السحب للتحديث** (Pull-to-refresh) وزر رجوع يتنقل داخل تاريخ تصفح الموقع بدل إغلاق التطبيق مباشرة.
- **صفحة عدم اتصال** بسيطة مع زر إعادة المحاولة عند انقطاع الإنترنت.
- الروابط الخارجية (واتساب، اتصال، بريد، مواقع أخرى) تُفتح بتطبيقات النظام بدل فتحها داخل التطبيق.
- أيقونة تطبيق مبدئية بلون بنفسجي مطابق لهوية لوحة تحكم مدير النظام (v4). يمكنك استبدالها لاحقًا.

## أسهل طريقة للحصول على ملف APK جاهز (بدون تثبيت أي برنامج)
المشروع يحتوي على GitHub Actions (`.github/workflows/build.yml`) يبني لك APK تلقائيًا:

1. أنشئ مستودع (Repository) جديد على GitHub وارفع محتوى هذا المجلد إليه.
2. بعد الرفع مباشرة، اذهب إلى تبويب **Actions** في المستودع — ستجد أن عملية البناء (Build APK) بدأت تلقائيًا.
3. بعد اكتمالها (يستغرق 3-5 دقائق)، افتح الـ run الأخير وستجد ملف **HRSawaedArab-debug-apk** في قسم Artifacts أسفل الصفحة — حمّله وفكّ الضغط عنه لتحصل على `app-debug.apk`.
4. انقل الملف لجوالك (عبر واتساب لنفسك، جوجل درايف، أو USB) وثبّته مباشرة (فعّل "السماح بالتثبيت من مصادر غير معروفة" إذا طلب منك النظام).

ملاحظة: هذا APK للتجربة (debug) وغير موقّع للنشر على المتجر. للنشر على Google Play تحتاج إصدار موقّع (راجع قسم "قبل النشر على المتجر" بالأسفل).

## أو: البناء يدويًا عبر Android Studio
1. افتح المجلد بواسطة **Android Studio** (Open an existing project).
2. اتركه يقوم بمزامنة Gradle تلقائيًا (Sync Now).
3. وصّل جهاز أندرويد حقيقي أو شغّل محاكي (Emulator)، ثم اضغط Run ▶️.
4. لإنشاء ملف APK للتوزيع: من القائمة العلوية اختر:
   `Build > Build Bundle(s) / APK(s) > Build APK(s)`
   وستجد الملف الناتج داخل: `app/build/outputs/apk/`

## قبل النشر على المتجر (اختياري)
- **معرّف الحزمة (Package name)**: حاليًا `com.sawaedarab.hr` — غيّره من `app/build.gradle` (`applicationId` و `namespace`) و مسار حزمة الكود إذا رغبت باسم مختلف.
- **الأيقونة**: الأيقونات الحالية بسيطة (حرف S بخلفية بنفسجية). لتصميم أيقونة احترافية: كليك يمين على `res` داخل Android Studio → `New > Image Asset` وارفع شعار الشركة.
- **شاشة البداية (Splash Screen)**: غير مضافة حاليًا؛ يمكن إضافتها لاحقًا عبر `SplashScreen API` إن رغبت.
- **الإشعارات (Push Notifications)**: غير مضمّنة؛ تحتاج Firebase Cloud Messaging إذا أردت إشعارات عند اعتماد طلبات، تنبيهات رواتب، إلخ.
- **التوقيع (Signing) للنشر على Google Play**: أنشئ Keystore من `Build > Generate Signed Bundle / APK`.

## ملاحظة أمنية
التطبيق يقيّد التنقل الداخلي على نطاق `hr.sawaedarab.com` فقط؛ أي رابط لنطاق آخر يُفتح خارج التطبيق تلقائيًا (متصفح/تطبيق مناسب) لحماية جلسة تسجيل الدخول.
